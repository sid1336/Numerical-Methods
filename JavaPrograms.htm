These programs were written by Richard Nunoo and give correct results for the examples and exercises in the text. However, no attempt was made to write a general-purpose professional software. Specially, the algorithms are not always in the form that leads to most efficient program in terms of time and storage requirements. 

You need a Java plug-in to run the applets, which can be downloaded from the Java website. Alternatively, you can download Netscape 7.1 which installs a Java plug-in automatically.
